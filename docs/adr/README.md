# Decisões de Arquitetura (ADR)

Registro das decisões técnicas e de produto. Uma decisão por arquivo, numerada.

## Formato

```
# XXXX — Título

**Data:** YYYY-MM-DD
**Status:** Aceita | Em discussão | Substituída por #XXXX

## Contexto
Por que a decisão precisou ser tomada.

## Decisão
O que foi decidido.

## Consequências
O que fica mais fácil, o que fica mais difícil, o que muda.
```
